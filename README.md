# Excel-Assignment-2-Data-Cleaning-and-Preparation
Simple Overview of Use/Purpose

This project is used to clean, transform, analyse, and visualize product data using Microsoft Excel.

Description

The project includes handling missing values, correcting inconsistent data, removing duplicate records, formatting dates and prices, and applying conditional formatting. The dataset contains product information such as manufacturing date, country code, product brand, price, quantity, and category.

Getting Started Dependencies

Requirements: • Microsoft Excel • Windows 10 or above
